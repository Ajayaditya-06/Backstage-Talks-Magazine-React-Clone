## Run Locally

Clone the project

```bash
  https://github.com/Ajayaditya-06/Backstage-Talks-Magazine-React-Clone.git
```

Go to the project directory

```bash
  cd "Backstage-Talks-Magazine-React-Clone"
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
